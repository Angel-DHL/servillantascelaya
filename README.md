# Servillantas Celaya - Sitio Web Profesional

Sitio web moderno y optimizado para SEO construido con React y Node.js.

## 🚀 Características

- ✅ Diseño responsive y profesional
- ✅ SEO optimizado con React Helmet
- ✅ Animaciones suaves con Framer Motion
- ✅ Formulario de contacto funcional
- ✅ Integración con Chatway
- ✅ Paleta de colores personalizada
- ✅ Optimizado para Google Search Console

## 📋 Requisitos Previos

- Node.js 14+ 
- npm o yarn
- Cuenta de Gmail para el envío de emails
